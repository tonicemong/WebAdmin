## 📝 프로젝트 소개
**WebAdmin**은 마인크래프트 서버 운영자가 게임 내에 접속하지 않고도 
<br>**웹 브라우저를 통해 실시간으로 서버를 관리**할 수 있도록 설계된 통합 관리 솔루션입니다. 

현 버전은 베타(BETA) 단계이므로 예상치 못한 오류가 발생할 수 있습니다.<br>
운영 중인 서버에 적용 전, 반드시 백업을 권장하며 발견된 문제점은 즉시 제보 부탁드립니다.

> "언제 어디서든 브라우저 하나로 당신의 서버를 컨트롤하세요."

---

## ✨ 주요 기능 (Key Features)

* **📊 실시간 서버 대시보드**
  * TPS, RAM 사용량, 온라인 플레이어 수 등 핵심 서버 상태를 실시간으로 확인합니다.

* **👥 플레이어 상세 관리**
  * 플레이어 IP, 권한, 게임모드, 체력, 허기, 레벨, 위치, 인벤토리 정보를 웹에서 조회합니다.
  * 상세창에서 경고, 채팅차단, Kick, Ban 등 주요 관리 작업을 수행할 수 있습니다.

* **💬 욕설 감지 및 자동 제재**
  * 욕설 감지 기록을 저장하고, 누적 횟수에 따라 경고 또는 채팅차단을 자동 적용합니다.
  * 자동 제재 룰은 웹에서 추가, 삭제, 활성화, 비활성화할 수 있습니다.
    
* **🧾 관리자 로그**
  * 로그인, 제재, 명령 실행, 공지, 배율 변경 등 주요 관리자 작업을 기록합니다.
  * 로그는 한글 분류, 대상, 내용, 시간 기준으로 표시됩니다.

* **💻 제한형 웹 콘솔**
  * 웹에서 서버 명령을 실행할 수 있으며, 허용된 명령어만 실행됩니다.

* **⏰ 핫타임 및 서버 배율 관리**
  * 경험치 배율, 드랍률 배율, 핫타임 보상 설정을 웹에서 관리합니다.

* **✅ 화이트리스트 / 밴리스트 관리**
  * 웹 UI에서 화이트리스트와 밴리스트를 조회하고 관리할 수 있습니다.

## 🚀 시작하기 (Getting Started)

### 1. 설치 방법
1. 본 저장소의 `Releases`에서 최신 `.jar` 파일을 다운로드합니다.
2. 마인크래프트 서버의 `plugins` 폴더에 파일을 넣습니다.
3. 서버를 실행하여 `config.yml` 파일을 생성합니다.
4. 생성된 `config.yml`을 수정 후 서버를 재시작 합니다.

### 2. 설정 안내
`plugins/WebAdmin/config.yml` 파일에서 다음 내용을 수정할 수 있습니다.
* **Port:** 웹페이지 접속 포트 (기본값: 8080)
* **Password:** 웹 로그인 시 사용할 비밀번호
